Sandook

Sandook is a command line task management utility for linux systems written using wonderful library 'urwid'. It can be run under cygwin on windows.

Requirements

Sandook requires

  • python 2.7
  • urwid

Installation

  • extract the downloaded sandook-x.x.tar.gz file
  • At the command prompt execute: python setup.py install. Admin privileges may be needed as per user's setup.
  • run sandook
  • Press ctrl+p to show a help dialog for a list of available commands
